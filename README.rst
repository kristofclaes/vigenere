========
Vigenère
========

.. image:: https://travis-ci.org/kristofclaes/vigenere.svg?branch=master
   :target: https://travis-ci.org/kristofclaes/vigenere
.. image:: https://codecov.io/gh/kristofclaes/vigenere/branch/master/graph/badge.svg
   :target: https://codecov.io/gh/kristofclaes/vigenere

``vigenere`` is a libary that allows you to do enciphering and deciphering using the Vigenère algorithm. Please note
that this is purely for educational purposes. The Vigènere algorithm is quite old and absolutely not safe to use in the
real world.
